# Aria - Mental Health Companion

An AI-powered mental health support chatbot with crisis detection, powered by Claude, ChatGPT, and Gemini.

## Setup

1. Clone the repo
2. Copy `.env.example` to `.env` and add your API key(s)
3. Run `npm install`
4. Run `npm start`
5. Open http://localhost:3000

## API Keys

You need at least one:
- **Claude**: Get from https://console.anthropic.com
- **ChatGPT**: Get from https://platform.openai.com
- **Gemini**: Get from https://aistudio.google.com

## Crisis Features

- Automatic crisis keyword detection
- Always-visible 988 Lifeline resources
- Safe messaging guidelines built into every AI response
